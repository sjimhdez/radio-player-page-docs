Radio Player Page — Documentation

Documentation for the Radio Player Page WordPress plugin. Available in three languages.

Documentation by language

  • Documentation (English – US) Full documentation in American English.
  • Documentación (Español – España) Documentación completa en castellano de España.
  • Documentação (Português – Brasil) Documentação completa em português do Brasil.

Each language folder contains an index and six topic pages: Installation, Configuration, Program Schedule, Visual Customization, Multiple Stations, and FAQ.
